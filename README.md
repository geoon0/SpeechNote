# SpeechNote Web

1. 프로젝트 소개

 ◉ 프로젝트명
   - SpeechNote Web (인공지능을 활용한 웹 기반 음성-텍스트 변환 서비스)

 ◉ 프로젝트 목적
   [프로그램 설명]
   • 웹 브라우저에서 음성을 텍스트로 변환하는 기능을 제공한다.
   • 별도의 프로그램 설치 없이 누구나 쉽게 사용할 수 있는 인터페이스를 구현한다.
   • 인공지능 API를 활용하여 실시간 음성 처리를 수행한다.
   [대상 사용자]
   • 빠르게 메모가 필요한 사용자 (녹음된 강의를 재학습) 
   • 음성 기반 인터페이스를 선호하는 사용자
   • 키보드 사용이 불편한 사용자 (접근성 지원 대상 포함)
   [기존 방법 대비 개선점]
   • 프로그램 설치 없이 웹에서 바로 사용 가능
   • 직관적인 UI를 통한 간편한 음성 입력/출력
   • 다양한 환경(PC, 모바일)에서 접근 가능
   • 인공지능 기반 음성 처리로 높은 편의성 제공

 ◉ 기대 효과
  • 인공지능 개념을 이해하고 활용 능력 향상
  • 웹 기반 서비스 개발 경험 확보
  • 음성 API 활용 능력 향상
  • 사용자 중심의 친화적인 인터페이스 설계 경험


2. 프로젝트 배경 및 범위

 ◉ 배경
   • 최근 비대면 환경과 모바일 사용 증가로 인해 음성 기반 인터페이스의 활용도가 점점 높아지고 있다. 특히 빠르게 정보를 기록해야 하는 상황에서 키보드 입력은 시간과 불편함을 초래할 수 있다.
또한, 기존 음성 인식 프로그램들은 별도의 설치가 필요하거나 특정 플랫폼에 제한되는 경우가 많아 접근성이 떨어지는 문제가 있다.
이러한 문제를 해결하기 위해, 별도의 설치 없이 웹 브라우저에서 바로 사용할 수 있는 음성-텍스트 변환 서비스를 개발하고자 한다. 이를 통해 사용자 편의성을 높이고, 인공지능 기술을 실생활에 활용하는 경험을 제공하는 것을 목표로 한다.


 ◉ 프로젝트 범위 (Scope)
  • 핵심 기능 A (음성 입력 및 변환) - 사용자의 음성을 입력받아 텍스트로 변환하는 기능
  • 핵심 기능 B (외부 API 연동) - 음성 인식을 위해 AI API를 연동하는 기능 (필수 요소)
  • 기본 UI/UX - 파일 업로드, 결과 표시 등 사용자 인터페이스
  • 고급 보안 처리 - 로그인, 암호화, 사용자 인증 등
  • 다국어 지원 - 여러 언어 음성 인식 기능
  • 데이터 저장 기능 - DB 저장, 사용자 기록 관리
  • 고급 AI 기능 - 요약, 문장 교정, 분석 등

 ◉ 사용 기술 스택
  • 언어: Java / JavaScript / HTML·CSS / Python 등
  • 프레임워크: Spring / SpringBoost
  • 버전 관리: Git &GitHub
