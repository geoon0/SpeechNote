# SpeechNote Web

## 1. 프로젝트 소개

### &nbsp; ◉ 프로젝트명
    - SpeechNote Web (인공지능을 활용한 음성-텍스트 변환 서비스)

### &nbsp; ◉ 프로젝트 목적
#### &nbsp; &nbsp;   [프로그램 설명]
#### &nbsp; &nbsp; &nbsp;    • 마이크로 녹음한 음성 또는 업로드한 음성 파일(mp3, wav 등)을 텍스트로 변환하여 화면에 출력
#### &nbsp; &nbsp; &nbsp;   • 사용 설명서 없이도 바로 쓸 수 있도록 버튼 중심의 단순한 화면 구성
#### &nbsp; &nbsp; &nbsp;  • 외부 음성 인식 API를 호출해 변환을 처리하고, 결과를 거의 실시간에 가깝게 확인 가능
#### &nbsp; &nbsp;   [대상 사용자]
#### &nbsp; &nbsp; &nbsp;  • 강의나 회의 녹음을 다시 듣기보다 텍스트로 빠르게 정리하고 싶은 학생·직장인
#### &nbsp; &nbsp; &nbsp;  • 타이핑보다 말로 설명하는 것이 더 익숙하거나 편한 사용자
#### &nbsp; &nbsp; &nbsp;  • 손이 자유롭지 못한 상황이나 키보드 사용이 어려운 접근성 지원 대상자
#### &nbsp; &nbsp;   [기존 방법 대비 개선점]
#### &nbsp; &nbsp; &nbsp;  • 최종 빌드본을 GitHub Releases에 JAR 파일로 올려 둘 예정이므로, Java 실행 환경(JRE)만 갖춰져 있으면 별도 설치 과정 없이 내려받아 바로 실행 가능
#### &nbsp; &nbsp; &nbsp;  • 녹음 → 변환 → 결과 확인까지의 흐름을 한 화면에서 끝낼 수 있도록 단순화한 UI
#### &nbsp; &nbsp; &nbsp;  • 규칙 기반 방식이 아닌 인공지능 모델을 활용하므로, 잡음이나 발음 변형이 있는 환경에서도 비교적 안정적인 인식 성능 기대

### &nbsp; ◉ 기대 효과
#### &nbsp; &nbsp; &nbsp;  • AI 모델이 음성 데이터를 어떻게 처리하는지 직접 다뤄 보며 인공지능 활용 능력 향상
#### &nbsp; &nbsp; &nbsp;  • 외부 API의 요청/응답 구조를 이해하고, Java HTTP 클라이언트로 직접 호출·연동해보는 실전 경험 축적
#### &nbsp; &nbsp; &nbsp;  • 사용자가 헷갈리지 않을 화면 흐름을 고민하면서 사용자 중심의 인터페이스 설계 경험 확보

## 2. 프로젝트 배경 및 범위

### &nbsp; ◉ 배경
#### &nbsp; &nbsp; &nbsp;  • 최근 비대면 강의·회의가 일상화되고 모바일 사용 시간이 늘어나면서, 음성으로 정보를 기록하고 전달하는 방식의 비중이 빠르게 커지고 있다. 특히 강의 중 메모, 회의 요약, 짧은 아이디어 기록처럼 즉시 정리가 필요한 상황에서는 키보드 입력이 속도가 느리고 손을 묶어두는 단점이 있다. 또한, 현재 시중에 나와 있는 음성 인식 프로그램은 대부분 별도 설치가 필요하거나 특정 OS·기기에 묶여 있어, 단순히 한두 번 변환하고 싶은 사용자 입장에서는 진입 장벽이 높은 편이다. 본 프로젝트에서는 이러한 불편을 줄이기 위해, JRE만 갖춰져 있으면 PC에서 곧바로 실행할 수 있는 자바 기반 음성-텍스트 변환 프로그램을 개발하고자 한다. 완성된 결과물은 GitHub Releases를 통해 JAR 파일 형태로 배포해 누구나 손쉽게 내려받아 사용할 수 있도록 하고, 이를 통해 사용자 편의성을 높이는 동시에 인공지능 기술을 단순한 데모 수준이 아니라 실제 사용 가능한 형태로 구현해보는 것을 목표로 한다.

### &nbsp; ◉ 프로젝트 범위
####  &nbsp; &nbsp; &nbsp; • 핵심 기능 A (음성 입력 및 변환) - 마이크 녹음과 음성 파일 업로드를 모두 지원하고, 변환된 텍스트를 화면에 출력
####  &nbsp; &nbsp; &nbsp; • 핵심 기능 B (외부 API 연동) - 음성 인식을 위해 AI(인공지능) API를 호출하고, 응답을 가공해 사용자에게 전달 (필수 요소)
####  &nbsp; &nbsp; &nbsp; • 기본 UI/UX - Swing 기반 창에서 파일 선택 영역, 변환 진행 상태 표시, 결과 텍스트 영역, 복사 버튼 등 최소 단위의 사용자 인터페이스
####  &nbsp; &nbsp; &nbsp; • 고급 보안 처리 - 로컬 로그인, 비밀번호 해시 저장, 사용자별 데이터 분리 등 (여건이 되면 진행)
####  &nbsp; &nbsp; &nbsp; • 다국어 지원 - 한국어 외 영어 등 여러 언어 음성 인식 기능
####  &nbsp; &nbsp; &nbsp; • 데이터 저장 기능 - 변환된 텍스트를 DB에 저장한 뒤 날짜·시간순으로 목록에 정렬해 보여 주고, 각 항목 옆에 요약·복사·다운로드·삭제 등 사용 가능한 기능을 함께 표기
####  &nbsp; &nbsp; &nbsp; • 고급 AI 기능 - 변환된 텍스트에 대한 요약, 맞춤법 교정, 키워드 추출 등 부가 기능

### &nbsp; ◉ 사용 기술 스택
#### &nbsp; &nbsp; &nbsp;   • 언어: Java (전반 로직 및 Swing 기반 GUI 구성) / SQL (DB 질의)
#### &nbsp; &nbsp; &nbsp;   • 라이브러리 · 도구: `java.net.http` (외부 음성 인식 API 호출), JDBC (DB 연결), Maven 또는 Gradle (빌드 도구)
#### &nbsp; &nbsp; &nbsp;   • 버전 관리 및 배포: Git & GitHub (브랜치 분리·코드 리뷰 기반 협업), 최종 빌드본은 GitHub Releases에 JAR 파일로 업로드해 배포
