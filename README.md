# SpeechNote Web

## 1. 프로젝트 소개

### 프로젝트명

SpeechNote Web — 인공지능을 활용한 음성-텍스트 변환 서비스

### 프로젝트 목적

**프로그램 설명**

- 마이크 녹음, 시스템 음성(스피커 출력) 동시 녹음, 음성 파일(mp3, wav 등) 업로드를 통해 음성을 텍스트로 변환·출력
- 화상통화·온라인 강의 등 컴퓨터로 진행되는 상황에서도 마이크 음성과 스피커로 출력되는 음성을 함께 기록 가능
- 사용 설명서 없이도 바로 쓸 수 있도록 버튼 중심의 단순한 화면 구성
- 외부 음성 인식 API를 호출해 변환을 처리하고, 결과를 거의 실시간에 가깝게 확인 가능

**대상 사용자**

- 강의·회의 녹음을 다시 듣기보다 텍스트로 빠르게 정리하고 싶은 학생·직장인
- 화상통화·온라인 강의 내용을 자동으로 받아 적고 싶은 사용자
- 타이핑보다 말로 설명하는 것이 더 익숙하거나 편한 사용자
- 손이 자유롭지 못하거나 키보드 사용이 어려운 접근성 지원 대상자

**기존 방법 대비 개선점**

- Zoom·Teams 같은 특정 회의 플랫폼에 묶이지 않고, 어떤 회의·강의 환경에서도 동일한 방식으로 사용 가능
- 마이크와 시스템 음성을 동시에 기록할 수 있어 화상통화·온라인 강의 내용까지 양쪽 음성 모두 정리 가능
- 변환 결과가 외부 서비스가 아닌 사용자 PC에 저장돼 자유롭게 가공·활용 가능
- 녹음 → 변환 → 결과 확인까지 한 화면에서 끝낼 수 있도록 단순화한 UI
- 인공지능 모델을 활용하므로 잡음이나 발음 변형이 있는 환경에서도 비교적 안정적인 인식 성능 기대

### 기대 효과

- AI 모델이 음성 데이터를 어떻게 처리하는지 직접 다뤄 보며 인공지능 활용 능력 향상
- 외부 API의 요청/응답 구조를 이해하고, Java HTTP 클라이언트로 직접 호출·연동해보는 실전 경험 축적
- 사용자 흐름을 고민하면서 사용자 중심의 인터페이스 설계 경험 확보

## 2. 프로젝트 배경 및 범위

### 배경

노트북·PC를 옆에 두고 회의·강의·인터뷰·스터디 등 다양한 자리에 참여하는 경우가 많아지면서, 음성으로 이뤄지는 내용을 그대로 기록하고자 하는 수요도 빠르게 늘고 있다. 강의 중 메모, 회의 요약, 짧은 아이디어 기록처럼 즉시 정리가 필요한 상황에서는 키보드 입력이 속도가 느리고 손을 묶어두는 단점이 있다. 노트북을 들고 참여하는 오프라인 미팅이나 강의실에서 수업 내용을 그대로 받아 적는 경우라면 마이크 음성만 잘 잡아도 충분하지만, 화상통화·온라인 강의처럼 컴퓨터로 진행되는 자리에서는 마이크로 들어오는 본인 목소리뿐 아니라 스피커로 출력되는 상대방·강사 음성까지 함께 기록할 수 있어야 실용적이다.

그러나 현재 쓸 수 있는 음성 인식 도구는 대부분 Zoom·Teams 같은 특정 회의 플랫폼이나 클라우드 서비스에 묶여 있어, 사용하는 프로그램이 달라지면 매번 다른 방식을 찾아야 하는 불편이 있다. 또한 마이크 음성과 컴퓨터에서 출력되는 시스템 음성을 동시에 잡아 주는 무료 도구는 흔치 않으며, 변환된 텍스트가 외부 서비스에 보관돼 따로 저장·가공하기 어려운 경우도 많다. 본 프로젝트에서는 이러한 한계를 보완하기 위해, 어떤 회의·강의 플랫폼을 쓰든 상관없이 PC에서 곧바로 실행해 마이크와 시스템 음성을 한꺼번에 기록·변환할 수 있는 자바 기반 음성-텍스트 변환 프로그램을 개발하고자 한다. 완성된 결과물은 GitHub Releases에 JAR 파일로 배포해 JRE만 갖춰져 있으면 누구나 사용할 수 있도록 하고, 변환 결과는 사용자 PC에 저장돼 본인이 자유롭게 활용할 수 있도록 한다. 이를 통해 인공지능 기술을 단순한 데모 수준이 아니라 실제 사용 가능한 형태로 구현해보는 것을 목표로 한다.

### 프로젝트 범위

- **핵심 기능 A (음성 입력 및 변환)**: 마이크 녹음, 시스템 음성(스피커 출력) 동시 녹음, 음성 파일 업로드를 지원하고 변환된 텍스트를 화면에 출력
- **핵심 기능 B (외부 API 연동)**: 음성 인식을 위해 AI API를 호출하고, 응답을 가공해 사용자에게 전달 (필수 요소)
- **기본 UI/UX**: Swing 기반 창에서 입력 소스 선택, 변환 진행 상태 표시, 결과 텍스트 영역, 복사 버튼 등 최소 단위의 사용자 인터페이스
- **데이터 저장 기능**: 변환된 텍스트를 DB에 저장한 뒤 날짜·시간순으로 정렬, 각 항목별 요약·복사·다운로드·삭제 기능 제공
- **로그인·보안**: 로컬 로그인, 비밀번호 해시 저장, 사용자별 데이터 분리 (여건이 되면 진행)
- **고급 AI 기능**: 변환된 텍스트에 대한 요약, 맞춤법 교정, 키워드 추출 등 부가 기능
- **다국어 지원**: 한국어 외 영어 등 여러 언어 음성 인식

### 사용 기술 스택

- **언어**: Java (전반 로직 및 Swing 기반 GUI), SQL (DB 질의)
- **라이브러리·도구**: `java.net.http` (외부 API 호출), JDBC (DB 연결), Maven 또는 Gradle (빌드)
- **버전 관리 및 배포**: Git & GitHub (브랜치 분리·코드 리뷰 기반 협업), 최종 빌드본은 GitHub Releases에 JAR 파일로 업로드
