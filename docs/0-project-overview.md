# 0. 프로젝트 개요

## 0.1 프로젝트 소개
- **프로젝트명**: SpeechNote — 데스크톱 음성 인식 메모장
- **목적**: 마이크와 시스템(스피커) 음성을 녹음하거나 오디오 파일을 업로드해 **STT로 텍스트 변환**하고, **AI 요약·키워드**까지 생성해 기록·관리하는 자바 데스크톱 애플리케이션.
  - **해결하려는 문제**: 화상회의·온라인 강의 음성을 사람이 일일이 받아 적어야 하는 번거로움.
  - **대상 사용자**: 회의록·강의 노트가 필요한 학생·직장인.
  - **기존 방법 대비 개선점**: 마이크뿐 아니라 **시스템 음성(상대방 목소리)** 까지 캡처하고, 결과를 로컬 DB에 보관하며 요약·자막(SRT)·문서(DOCX)로 내보낼 수 있음.
- **기대 효과**: 회의·강의 내용을 자동으로 텍스트·요약으로 남겨 기록 부담을 줄임. 자바 GUI(Swing)·DB(JDBC)·외부 AI API 연동·오디오 캡처를 통합 경험.

## 0.2 배경 및 범위
- **배경**: 순수 자바 환경에서 음성→텍스트 파이프라인을 직접 구성. STT/LLM은 외부 API, 오디오 캡처는 Java Sound API를 활용.
- **범위 (Scope)**

  | 구분 | 내용 |
  | --- | --- |
  | **In-Scope** | 마이크·시스템 녹음, 파일 업로드 변환, 기록 DB 저장·검색·정렬, TXT/DOCX/SRT 내보내기, 회원가입·로그인, AI 요약·키워드 |
  | **Out-of-Scope** | 실시간 스트리밍 변환, 다중 기기 동기화(중앙 DB 서버), 고급 보안(salt·2FA), 모바일 |

- **기술 스택**: Java 11 · Maven · Swing/FlatLaf · SQLite(JDBC) · java.net.http · org.json · Apache POI · Git/GitHub
